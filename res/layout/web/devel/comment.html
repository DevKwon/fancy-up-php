    댓글 설치 방법

    1) 스크립트
                <!--COMMENT-->
            <link href="/modules/comment/skin/devel/style.css" rel="stylesheet"><!--comment skin-->
            <script type="text/javascript" src="/modules/comment/js/comment.js"></script>
            <script type="text/javascript">
            
            //comment
            var act = '{%params.act%}';
            var comment_id='notice';
            var comment_skin = 'devel';
            var comment_foreign_key = '';
            
            // paging
            var paging_theme = 'gray';
            
            $(document).ready(function()
            {
                comment_skin_template('list',{'foreign_key':comment_foreign_key,'page':1});
                comment_skin_template('write',{'foreign_key':comment_foreign_key});
            });
    2) HTML
      3) 설정 관련 모듈(XML)

      식별자 설명

      • @comment_id : 댓글이 달리는 게시판 코드명
      • @comment_skin : 댓글 리스트/쓰기 스킨 폴더명(comment/skin/devel)
      • @comment_foreign_key : 게시물 코드번호(게시판에서의 게시물 번호)
      • @paging_theme : 페이징 네이게이션 css 테마(스킨)명
      • @comment_skin_template : 댓글 탬플릿 호출 자바스크립트 함수(실행모드, 게시판/게시물 정보)
